Soil Mechanics Report


In progress — development phaseEn curso — fase de desarrollo

A web application for automating geotechnical report generation. The system allows users to structure, manage and consolidate information from multiple sources into a standardized document. It optimizes the workflow through metadata configuration, section selection and data linking from SharePoint repositories — reducing manual intervention and enabling version management and document delivery. Una aplicación web para la automatización de la generación de reportes geotécnicos. El sistema permite a los usuarios estructurar, gestionar y consolidar información proveniente de múltiples fuentes en un documento estandarizado. Optimiza el flujo de trabajo mediante configuración de metadata, selección de secciones y vinculación de datos desde SharePoint — reduciendo la intervención manual y facilitando la gestión de versiones y entrega de documentos.
